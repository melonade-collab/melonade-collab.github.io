Agentic AI

Intake & Triage Agent

Agentic AI

Conversational Feedback Agent

AI prototype

Natural Language Shipping Form ★ Favourite

Shipping forms demand precise field-by-field input that creates friction for occasional shippers. Explores replacing structured entry with plain language, letting AI parse the description and pre-populate the form.

Re-imaginings

B2B mobile

Enterprise Mobile Self-Service

Corporate mobile subscribers had no visibility into usage — overages only surfaced on the company bill. Built a SIM-authenticated app surfacing usage, plan details, and account tasks.

Web accessibility

Accessibility Remediation

Unowned accessibility debt blocked assistive technology users site-wide. Audited violations, coordinated cross-team fixes, and embedded tooling to reach WCAG 2.1 AA compliance.
